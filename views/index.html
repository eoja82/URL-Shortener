URL Shortener Microservice

Why? A long URL distracts readers from your message. Also, in situations where there are character liimts, a long URL uses too many characters.

Create a short URL

example: POST [project_url]/api/shorturl/new - https://www.google.com

Example Usage:

https://cobalt-barracuda.glitch.me/api/shorturl/1

Will Redirect to:

https://www.google.com
